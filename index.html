Dashboard "Localization and Date-Time Formatting"

The dashboard contains two gantt charts: the first one is project chart and the second one is resource chart.

Interactivity

Using Localisations:

  1. Choose desirable language from the right menu
  2. Choose Date and Time format for your language
  3. If you want a customized Date-Time format - use the text field below the list of formats
